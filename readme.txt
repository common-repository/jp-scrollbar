JP Scrollbar is a WordPress plugin that enables a awesome scrollbar instead default scrollbar.


== Description ==

JP Scrollbar is a wordpress plugin for your wordpress website. You can easily replace your default scrollbar with a nice looking scrollbar.
To use this plugin you need to install the plugin and activate. Easy enough!

You can change your scrollbar theme, scroll speed, autohide mode, keyboard scrolling etc.

== Installation ==

Installing this plugin as regular wordpress plugin.

After install, you have to activate the plugin. Then you will find the awesome scrollbar to right of your website.


Now visit your website & see something awesome. :)

== Frequently Asked Questions ==

= How can I change scrollbar theme? =
Go to your admin panel. Click JP Scrollbar options submenu under setting menu. Choose your theme from dropdown menu. Finally click on submit.


= How can I change scrollbar color? =
This option is not available now.


= Have you any shortcode for this plugin? =
No, no need to use shortcode for this plugin. Install, activate & enjoy the plugin.
